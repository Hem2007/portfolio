# Hemnath V - Professional Portfolio Website

A modern, responsive, and professionally designed portfolio website showcasing my skills in **MERN Stack Development**, **Data Analytics**, and **AI/Machine Learning**.

## 🌟 Key Features

- ✅ **Multi-Page Structure** - Home, Skills, Resume, and Connect sections
- ✅ **Fully Responsive Design** - Optimized for desktop, tablet, and mobile devices
- ✅ **Dark/Light Theme Toggle** - User preference saved in localStorage
- ✅ **Professional UI/UX** - Modern design with smooth animations
- ✅ **Skill Cards with Progress Bars** - Visual representation of technical expertise
- ✅ **Professional Resume Layout** - Two-column design with experience, education, and projects
- ✅ **Fast Performance** - Optimized for speed and SEO
- ✅ **Smooth Animations** - Scroll-triggered fade-in effects and hover transitions

## 🚀 Getting Started

### For Local Development

1. Clone the repository:
```bash
git clone https://github.com/Hem2007/portfolio.git
cd portfolio
```

2. Open in a local server (optional):
```bash
# Using Python
python -m http.server 8000

# Or using Node.js simple-http-server
npm install -g simple-http-server
simple-http-server
```

3. Open in your browser:
```
http://localhost:8000
```

### Customization

- **Update Profile Image**: Replace `assets/profile.jpg` with your image
- **Update Resume**: Replace `assets/resume.pdf` with your resume
- **Edit Content**: Modify HTML files to update personal information
- **Change Colors**: Update CSS variables in `styles.css`:
  ```css
  --primary-color: #2563eb;
  --accent-color: #10b981;
  ```
